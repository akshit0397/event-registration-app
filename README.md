# College Event Registration System

## Project Overview

This is a web application developed using Flask and SQLite that allows students to register for college events. The application collects user details through a form and stores them in a database. It also provides a page to view all registrations.

## Objective

* To create a simple and user-friendly event registration system
* To store and manage data using a database
* To demonstrate backend development using Flask

## Technologies Used

* Python (Flask)
* HTML/CSS
* SQLite Database

## Installation Steps

### 1. Clone the repository

git clone https://github.com/akshit0397/event-registration-app.git
cd event-registration-app

### 2. Install dependencies

pip install flask

### 3. Run the application

python app.py

### 4. Open in browser

http://127.0.0.1:5000/

### 5. View stored records

http://127.0.0.1:5000/records

## Database Details

* Database used: SQLite
* File name: registrations.db
* Table name: registrations

## Features

* User registration form
* Data stored in database
* Success page after submission
* View all records using /records route
